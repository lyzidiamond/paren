rpl-www playground

rpl-www is an environment for JavaScript live-coding in browsers. Try it out below: write JavaScript as you would usually, and add magic comments, starting with //= to inspect values as you go.

Source Code & Documentation on Github

var pt = turf.point(0, 0); //=pt var a = 1; var c = a + 1; //=c function addTwoNumbers(a, b) { //=a //=b return a + b; } var result = addTwoNumbers(10, 32); //=result
